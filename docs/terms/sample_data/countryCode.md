# Term: countryCode

*Recommended best practice is to use an ISO 3166-1-alpha-2 country code. Recommended best practice is to use XZ if outside the EEZ (i.e. open seas). Lookup codes here:
https://www.iso.org/obp/ui/#search*

Origin: AOML

Example: US
